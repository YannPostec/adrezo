Alimentation de la base adrezo

Les données proviennent de Cacti, plusieurs tâches planifiées permettent d'alimenter la base :
  • quotidiennement, la base adrezo est remplie avec les evolutions du parc Cacti (CactiDevicesJob).
    Les nouveaux éléments sont ajoutés, ceux devenus inexistants sont marqués comme Détruit mais reste pour les statistiques, ceux marqués Disable dans Cacti le deviennent également dans la base adrezo.
  • toutes les heures, jours ou mois selon ce qui est défini pour CactiStatsJob, les stats de Cacti sont reportés dans adrezo et sont réinitialisés dans la base Cacti.
  • les données sont aggrégées des heures en jours et des jours en mois (CactiAggregateHoursJob, CactiAggregateDaysJob) pour former les statistiques mensuelles

Avec les droits nécessaires, vous pouvez gérer les différents items de la base adrezo pour ces statistiques

Concept global des pages

Vous trouverez dans chaque page deux zones :
  • une zone d'ajout
  • la liste des éléments existants avec leurs informations
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un élément en cliquant sur
  • Modifier un élement en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher

Gestion des Clients

Attention si vous supprimez un client : suppression des sites associés

Un client est défini par :

  • son nom
  • si on veut qu'il soit visible ou non dans les statistiques
  • le planning associé (tout changement modifie les sites et équipements associés)

Gestion des sites

Attention si vous supprimez un site : Les équipements associés seront remis dans le client/site Inconnu
Les sites sont un "nom" générique pour entité/zone/regroupement d'équipement.

Un site est défini par :

  • son client
  • son nom
  • si on veut qu'il soit visible ou non dans les statistiques
  • le planning associé (tout changement modifie les équipements associés)

Gestion des équipements (devices)

Les equipements sont importés dans adrezo par les tâches planifiées.
Ils sont définis pour leur premier ajout dans un site spécial "Inconnu" pour le client "Inconnu".
Ces clients/sites ne s'affichent pas dans les statistiques, il faut donc passer sur cette page pour mettre à jour la liste des équipements.

Un équipement est défini par :

  • son site/client
  • son nom
  • son ID de la base Cacti à des fins de debug
  • son status, les valeurs possibles sont
    • 0 : Inactif (correspond à Disable dans Cacti)
    • 1 : Actif (correspond à Enable dans Cacti)
    • 2 : Detruit (n'existe plus dans Cacti)
  • le planning associé

Gestion des plannings

Attention si vous supprimez un planning : les clients,sites,équipements associés seront remis dans le planning par défaut (tout le temps)
Les plannings définissent des périodes de temps ou les statistiques ne seront pas collectés.

Une fois un planning créé, vous pouvez sélectionner les heures (de 0h à 23h) où la collecte ne s'effectuera pas en cliquant sur
Chaque heure X représente de Xh00 à Xh59.
Procédez ainsi pour chaque jour de la semaine, vous avez également la possibilité de sélectionner par jour toutes les heures, aucun ou d'inverser la sélection.
Vous pouvez ensuite valider avec

Un planning est défini par :

  • son id
  • son nom
  • le choix des heures où la collecte ne s'effectuera pas

Vous pouvez également copier un planning en cliquant sur , une fenêtre vous demande de rentrer le nom de ce nouveau planning.
La création se fait en cliquant sur

Nettoyage des Statistiques

Sur la page Purge, vous pouvez voir le nombre de lignes contenues dans la table statistiques.
Ces lignes apparaissent par mois, le mois étant nommé <YYYYDD> (YYYY : année, MM : mois)
Pour supprimer les statistiques d'un mois, il vous suffit de cliquer sur .
