Redondance réseau

Avec les droits nécessaires, vous pouvez gérer les redondances.

Gestion des protocoles

Vous trouverez dans la page deux zones :
  • une zone d'ajout
  • la liste des protocoles existants avec leurs informations
Pour chaque ligne du tableau, vous pouvez :
  • Effacer un protocole en cliquant sur . Attention cela entraine la suppression des redondances associés
  • Modifier un protocole en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.

Un protocole est défini par un seul champ :

  • son nom

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher

Gestion des redondances

Vous trouverez dans la page deux zones :
  • une zone d'ajout
  • la liste des protocoles existants avec leurs informations
Pour chaque ligne du tableau, vous pouvez :
  • Effacer une IP de redondance en cliquant sur
  • Modifier une IP de redondance en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.

Une redondance est définie par :

  • son protocole
  • son ID
  • l'adresse IP de la redondance

L'adresse IP est choisie en cliquant sur , ce qui ouvre un popup vous permettant de faire une recherche par le nom ou par l'ip.
Les sélections se font en cliquant sur

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher
