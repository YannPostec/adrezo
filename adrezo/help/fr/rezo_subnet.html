Gestion des Subnets

Avec les droits nécessaires, vous pouvez gérer les subnets.

Vous trouverez dans la page deux zones :
  • une zone d'ajout
  • la liste des subnets existants avec leurs informations
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un subnet en cliquant sur . Attention cela entraine la suppression de tous les objets associés
  • Modifier un subnet en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.
Un subnet est défini par plusieurs champs :
  • le site de rattachement
  • l'ip de depart du subnet
  • le masque de sous-reseau
  • son libellé
  • l'adresse IP de la passerelle (pour information à l'utilisateur)
  • l'adresse IP de broadcast (pour avoir une borne finie dans la recherche d'IP dispo)
  • le vlan, représenté par son ID, auquel est rattaché ce subnet. Si aucun, choisir l'ID 0 qui signifie 'Pas de vlan'
L'adresse IP de broadcast est automatiquement calculé quand vous entrez l'ip et le masque du subnet.
Une vérification si le subnet entré est bien une adresse de subnet est effectuée en cas d'ajout ou de modification.

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher
