Gestion de la norme

Avec les droits nécessaires, vous pouvez gérer les surnets définissant la norme.
Utilisez le bouton pour entrer ou sortir du mode admin.

Sur chaque surnet, vous trouverez alors 4 icônes :

  • Supprimer le surnet
  • Ajouter un surnet fils à ce surnet
  • Modifier les informations de ce surnet
  • Gérer la liste de subnets associés à ce surnet
  • Calculer les subnets disponibles dans ce surnet

Supprimer

Cliquer dessus pour faire apparaitre une fenetre de confirmation
Attention, cela supprime tous les surnets fils en cascade, ainsi que tous les liens avec les subnets du surnet ou des fils. Cependant les subnets ne sont en rien modifiés

Ajouter

Cliquer dessus pour faire apparaitre un popup vous permettant de rentrer les informations du surnet fils :
  • son ip
  • son masque
  • son libellé
  • ses informations avec un éditeur
  • si le surnet doit proposer le calcul de subnets disponible
Valider la création en cliquant sur
Si l'ip/mask entrés ne forment pas une adresse de subnet valide, un message vous avertit.
Il est cependant possible de valider l'ajout.

Modifier

De la même façon que pour l'ajout, un popup vous permet de modifier les champs du surnet

Liste Subnets

Cette liste présente les subnets rattachés au surnet.
Dans la partie droite se trouve la liste des subnets rattachés.
Dans la partie gauche se trouvent les subnets que vous pouvez rattacher.
La partie gauche est remplie à chaque fois que vous choisissez un contexte, vous pouvez affinez en sélectionnant un site.
Il est possible de bouger un ou plusieurs éléments entre les listes en utilisant les icônes :
  • Passer de la colonne de gauche à celle de droite
  • Passer de la colonne de droite à celle de gauche
Il vous suffit de valider en cliquant sur pour attacher au surnet l'ensemble des subnets sélectionnés.

Calcul des Subnets disponibles

Dans le popup qui s'ouvre, il vous suffit de rentrer le masque des subnets que vous rechercher et l'application renverra une liste des subnets disponibles.
