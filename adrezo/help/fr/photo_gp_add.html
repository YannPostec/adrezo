Ajout d'une photo

En vue admin :

  • pour les photos de la salle, cliquer sur
    Puis dans la fenetre qui s'ouvre, rentrez un nom (obligatoire) dans le champ approprié
  • pour les baies, cliquez sur qui se trouve à côté du nom de chaque baie.
    Choisissez la position de la photo parmi la liste proposée.
    La liste n'est composée que des élements (Recto Haut, Recto Bas, Verso Haut, Verso Bas) qui ne sont pas encore attribués.
    Si vous voulez positionnez la photo sur un emplacement deja occupé, il vous faut remplacer la photo existante.

Dans les deux cas, selectionnez un fichier image et cliquer sur pour telecharger votre image sur le serveur.
Pour les navigateurs supportant HTML5, une barre de progression apparait.
Une fois l'upload terminé, un message vous prévient que le traitement (création de thumbnails et stockage dans la base de données) est en cours.
Une fois le traitement terminé, un bouton apparait
