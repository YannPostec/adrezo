Remplacement d'une photo

En vue admin, un icône se trouve à côté de chaque photo.
Cliquez dessus pour faire apparaître une fenêtre vous permettant de remplacer la photo.

Selectionnez un fichier image et cliquer sur pour telecharger votre image sur le serveur.
Pour les navigateurs supportant HTML5, une barre de progression apparait.
Une fois l'upload terminé, un message vous prévient que le traitement (création de thumbnails et stockage dans la base de données) est en cours.
Une fois le traitement terminé, un bouton apparait
