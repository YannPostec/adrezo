Gestion des baies

Avec les droits nécessaires, vous pouvez gérer les baies d'un ensemble donné.

Vous trouverez dans la page deux zones :
  • une zone d'ajout
  • la liste des baies existantes avec leurs informations
Pour chaque ligne du tableau, vous pouvez :
  • Effacer un baie en cliquant sur . Attention cela entraine la suppression de tous les objets associés
  • Modifier une baie en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.
Une baie est définie par plusieurs champs :
  • son site
  • sa salle
  • son ensemble de rattachement
  • son nom
  • la position de la baie dans l'ensemble

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher
