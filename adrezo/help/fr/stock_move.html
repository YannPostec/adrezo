Mouvements entre Stocks

Avec les droits nécessaires, vous pouvez déplacer une quantité d'un type dans un stock vers un autre type d'un autre stock.
Et ceci indépendamment du contexte, cependant seuls les contextes ou vous avez les droits de manipuler les stocks sont possibles.

Pour se faire, sélectionner les types source et destinations, le stock de chacun est affiché dans la colonne Stock.
Rentrez la quantité à déplacer et utiliser l'icône pour valider.
