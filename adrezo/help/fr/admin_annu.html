Gestion des Annuaires

Avec les droits nécessaires, vous pouvez gérer les annuaires.
Cependant il n'est pas possible d'altérer l'annuaire Local prédéfini.

Vous trouverez dans la page un tableau avec :
  • une zone d'ajout en première ligne
  • la liste des annuaires existants avec leurs informations sur les lignes suivantes
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un utilisateur en cliquant sur
  • Modifier un utilisateur en cliquant sur
En cas d'ajout ou de modification, les icônes apparaissent.
Entrez ou modifiez les différentes informations puis cliquez sur pour valider.
Vous pouvez également cliquer sur pour annuler, tous les champs reviennent à leur valeur initiale.
Un annuaire est défini par plusieurs champs :
  • un nom
  • un ordre qui déterminera l'ordre de recherche lors de l'authentification des utilisateurs
  • son type dans une liste prédéfinie :
    • Local : La base locale d'utilisateurs. Elle existe déjà et ne peut être modifié ou supprimé.
    • LDAP : Le seul type aujourd'hui disponible. Choisir ce type pour interroger un annuaire distant LDAP ou dérivé (tel qu'Active Directory)
    • D'autres types tels que Radius ou TACACS pourraient s'ajouter à cette liste.
  • la configuration pour les annuaires externes

Configuration LDAP

La configuration LDAP n'est pas disponible directement à la création de l'annuaire. Elle se fait une fois ce dernier créé
La fenêtre de configuration contient plusieurs champs à remplir, certains sont obligatoires en gras.
D'autres prendront les valeurs par défaut si vous les laissez vide.
Un texte explicatif en italique est présent en dessous de chaque champ, merci de vous y reporter pour les explications.

N.B. : Par mesure de sécurité, le mot de passe n'est jamais affiché
