Gestion des Types de consommables

Cette page ne s'affiche que si un stock principal a été défini pour le contexte.

Avec les droits nécessaires, vous pouvez gérer les types de consommables.
Vous trouverez dans la page deux zones :

  • une zone d'ajout
  • la liste des types existants avec leurs informations
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un type en cliquant sur . Attention cela entraine la suppression de son historique
  • Modifier un type en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.
Un type est défini par plusieurs champs :
  • le site de rattachement
  • la catégorie de rattachement
  • l'index correspondant au bac dans le stock
  • son libellé
  • le seuil d'alerte où un mail sera envoyé au gestionnaire du stock
  • le nombre d'élements en cours de commande
Le seuil est déclenché quand le stock plus la commande en cours sont inferieurs à ce seuil.

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher
