Gestion des Serveurs DHCP

Avec les droits nécessaires, vous pouvez gérer les serveurs DHCP utilisé par la tache planifiée de collecte des informations DHCP.

Vous trouverez dans la page deux zones :
  • une zone d'ajout
  • la liste des serveurs existants avec leurs informations
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un serveur en cliquant sur
  • Modifier un serveur en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.
Vous pouvez également gérer par serveur des scopes qui ne seront pas traités lors de la collecte d'informations. Pour cela :
  • Ajouter une exclusion en cliquant sur
  • Lister et supprimer les exclusions en cliquant sur
La suppression se fait en cliquant sur
Lors de l'ajout, la liste des scopes du serveur vous est proposé moins ceux déjà exclus, vous pouvez ajouter une exclusion en cliquant sur près du scope voulu.
Vous pouvez tester la connexion à un serveur DHCP en cliquant sur

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher

Un serveur est défini par plusieurs champs :
  • le type de serveur DHCP
  • le nom ou l'ip du serveur
  • le port utilisé pour atteindre l'API sur le serveur
  • la méthode d'accès SSL (HTTPS) ou non (HTTP)
  • l'utilisation ou non de l'authentification basique web pour se connecter
  • si authentification, le login utilisé
  • si authentification, le mot de passe utilisé (le mot de passe n'est jamais affiché)
  • la liste de ses scopes exclues, si existante
