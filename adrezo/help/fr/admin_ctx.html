Gestion des Contextes

Avec les droits nécessaires, vous pouvez gérer les contextes

Vous trouverez dans la page un tableau avec :
  • une zone d'ajout en première ligne
  • la liste des contextes existants avec leurs informations sur les lignes suivantes
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un contexte en cliquant sur . Attention cela entraine la suppression de tous les objets associés
  • Modifier un contexte en cliquant sur
En cas d'ajout ou de modification, les icônes apparaîssent.
Entrez ou modifiez les différentes informations puis cliquez sur pour valider.
Vous pouvez également cliquer sur pour annuler, tous les champs reviennent à leur valeur initiale.
Un contexte est défini par un seul champ :
  • son libellé

N.B :

  • Le dernier contexte dans la liste ne peut jamais être supprimé.
  • Lors de la création d'un nouveau contexte, il est impératif de se délogguer et se relogguer à l'application afin de récupérer les bons droits par défaut dessus.
