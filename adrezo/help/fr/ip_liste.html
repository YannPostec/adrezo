Cette liste propose une visualisation des enregistrements. Un enregistrement comporte les champs suivants :

  • Type : Adresse IP statique ou dynamique. Valeurs "static" (rentrée par un utilisateur), "dynamic" (crée par l'application) et "dhcp" (crée par le script des serveurs DHCP)
  • Site : Nom du site
    Le passage de la souris sur le site permet de visualiser le code site.
  • Nom : Nom normé de l'équipement
  • Libelle : Descriptif de l'équipement
  • Adresse IP : <Adresse IP>/<Masque de Sous-Réseau au format CIDR>
    Le passage de la souris sur l'adresse IP permet de visualiser le masque au format standard.
  • Sous réseau : Nom du Sous-réseau auquel l'adresse IP appartient
    Le passage de la souris sur le sous-réseau permet de visualiser sa passerelle.
  • Adresse MAC : Chaîne de 12 caractères représentant l'adresse MAC
  • Dernière Modification : Tracabilité de la dernière modification par <login> aux <date> <heure> indiquées
  • Temporaire : définit si l'adresse IP est affectée temporairement, si oui qui a affecté cet état et jusqu'à quand.
    Les valeurs possibles sont "NON" ou "OUI (<login>) jusqu'au <date>"
  • Migration : définit si l'adresse IP doit migrer, si oui qui a affecté cet état et pour quand.
    Les valeurs possibles sont "NON" ou "OUI (<login>) le <date> en <future adresse IP/masque> (<futur site>/<futur subnet>)"

Les différentes actions Effacer, Modifier et Migrer constituent les 3 premières colonnes de cette liste.
L'utilisateur ne possédant pas les droits nécessaires n'a pas accès à ces colonnes

Pour les enregistrements de type "dhcp", les plages d'adresses délivrées en DHCP sont rentrées dans la base avec le nom "RESA_DHCP" et le libellé "Reservation DHCP".
On trouve également :

  • Les réservations statiques d'adresse IP rentrées avec le nom de la station en Nom, son adresse MAC et le Libellé "Reservation Statique DHCP"
  • Les affectations dynamiques d'adresse IP rentrées avec le nom de la station en Nom, son adresse MAC, le Libellé "Adresse DHCP" et la date de resiliation du bail en Date Temporaire

Si la fonction est activée, en cliquant sur l'adresse IP, une fenêtre s'ouvre avec des urls contruite autour de cette adresse IP, vous pouvez ensuite ouvrir une nouvelle fenêtre allant à l'adresse indiquée en simplement cliquant sur une des url.

Si la fonction est activée, en cliquant sur l'adresse MAC, une page s'ouvre sur le site www.macvendors.com pour retrouver le code vendeur de cette adresse MAC.
