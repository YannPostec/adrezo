Formulaire de recherche d'adresses IP disponibles

Ce formulaire permet de rechercher un nombre d'adresses IP disponibles dans un sous-réseau en commencant ou non la recherche à partir d'une certaine adresse IP.

Le champ nombre d'IP disponible est obligatoire et est par défaut à 1.
Le champ sous-réseau est obligatoire.
Le champ donnant l'adresse IP de départ est optionnel. Par défaut, la recherche commence au début du sous-réseau spécifié.
Si elle est donnée, l'adresse IP de départ doit appartenir au sous-réseau sélectionné.

La recherche se fait en cliquant sur

Si des IP sont disponibles, le résultat se présente sous la forme d'un tableau avec:

  • l'adresse IP
  • la possibilité d'envoyer l'adresse dans une des lignes du tableau d'ajout/modification
  • la possibilité d'envoyer l'adresse dans le champ migration d'une des lignes du tableau d'ajout/modification si la migration est coché

L'ensemble des adresses de la recherche peuvent :

  • être ajoutées aux lignes du tableau d'ajout/modification
  • remplacer complètement le tableau d'ajout/modification
Ces actions efface la recherche une fois terminée.

Vous pouvez également effacer la recherche en cliquant sur ou rechercher à nouveau ce qui écrasera la recherche en cours
