Gestion des Mails

Avec les droits nécessaires, vous pouvez modifier le contenu des mails envoyés par l'application.

Vous trouverez dans la page une liste des mails avec :
  • le numéro du mail type
  • la langue du mail type sous forme d'icône
  • la table de la base de données utilisé pour créer les variables
  • le destinataire du mail
  • le sujet du mail
  • le message du mail
Dans le sujet et le message du mail, des variables peuvent être placés qui seront remplacés par les valeurs de la table indiquée.
Les noms de variables sont les noms de colonnes de la table.
Ces variables doivent être mis entre deux caractères #. Exemple : #IDX# permet d'afficher la valeur de la colonne IDX.

Pour certaines entrées, le destinataire, le sujet ou le message peuvent être remplacés par USERDEF et seront gérés par l'application
Pour certains mails types tel que l'envoi de mail aux utilisateurs, la table ne sera pas utilisée

Après modification des paramètres, cliquer sur pour valider.
