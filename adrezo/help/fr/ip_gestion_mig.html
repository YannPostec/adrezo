Migration d'adresses IP

Dans la liste des adresses IP se trouve la colonne "Migrer".
Dans cette colonne et sur les lignes ou il existe une IP de migration se trouve une case à cocher
Sélectionnez les enregistrements à migrer en cochant ces cases.
Dans l'entète de la colonne se trouve un lien Tous permettant d'inverser la sélection effectuée.

Cliquez ensuite sur le bouton ce qui fera apparaîre une fenêtre de confirmation.
Après confirmation, les enregistrements sélectionnés seront migrés et la liste d'adresses sera rechargée.
