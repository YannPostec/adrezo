Modification d'adresses IP

Dans la liste des adresses IP se trouve la colonne "Modifier".
Dans cette colonne et sur toutes les lignes se trouve une case à cocher
Sélectionnez les enregistrements à modifier en cochant ces cases.
Dans l'entète de la colonne se trouve un lien Tous permettant d'inverser la sélection effectuée.

Cliquez ensuite sur le bouton ce qui complètera le tableau ajout/modification
Voir la gestion de l'ajout pour les explications sur cette zone.
