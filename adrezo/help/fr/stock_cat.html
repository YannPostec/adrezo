Gestion des Catégories de consommables

Avec les droits nécessaires, vous pouvez gérer les categories.

Vous trouverez dans la page un tableau avec :
  • une zone d'ajout en première ligne
  • la liste des catégories existantes avec leurs informations sur les lignes suivantes
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un catégorie en cliquant sur . Attention cela entraine la suppression de tous les objets associés
  • Modifier un catégorie en cliquant sur
En cas d'ajout ou de modification, les icônes apparaîssent.
Entrez ou modifiez les différentes informations puis cliquez sur pour valider.
Vous pouvez également cliquer sur pour annuler, tous les champs reviennent à leur valeur initiale.
Une catégorie est défini par un seul champ :
  • son nom
