Administration de l'application

Avec les droits necessaires, vous avez acces aux paramètres de l'application.
Ils sont composés de :

  • Langue par défaut de l'application
  • Contexte par défaut de l'application
  • Gestion du compte admin
  • Gestion des URL pour les adresses IP
  • Outils de maintenance

Choix par défaut

Sélectionner la langue ou le contexte par défaut en cliquant dessus.
La langue ou le contexte actuel apparaissent en vert.

Compte admin

Ici vous pouvez changer le mot de passe et/ou le mail du compte admin.
Le mot de passe n'est jamais affiché.

Gestion des URLS

Chaque url est construite avec <protocole>://<IP>:<port>/<URI>
Le port et l'URI ne sont pas obligatoires, si vous rentrez le port 80 pour le protocole http ou 443 pour https, les ports ne seront pas pris en compte.

Lien entre comptes normal et exploitation

En authentification externe, si les comptes utilisées sont des comptes d'exploitation et ne contiennent pas d'adresse mail,
Et si le login de ces comptes est dérivé du compte normal utilisateur qui lui contient une adresse mail,
Vous pouvez lier cette information en mettant le prefixe et/ou suffixe utilisé pour retrouver le login du compte normal.

Outils de maintenance

Voir les logs de l'application qui sont :

  • photo : les erreurs liés au traitement des photos
  • database : les erreurs liés à la base de données
  • adrezo-default : le log de l'application
  • adrezo-api : le log de l'api
  • adrezo-beans : le log des beans
  • adrezo-jsp : le log des taglib jsp
  • adrezo-servlets : le log des servlets
  • adrezo-listener : le log des listeners
  • quartz : le log du module quartz gérant les tâches planifiées
  • quartz-jobs : le log des tâches planifiées

Gestion du logo

Vous pouvez changer le logo de la page de login en important une image au format PNG.
Il est également possible de revenir au logo par défaut en cliquant sur le bouton approprié.
