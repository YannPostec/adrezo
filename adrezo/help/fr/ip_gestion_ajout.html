Ajout d'adresse IP

Les champs obligatoires sont Nom et Adresse, il est préférable de remplir un libellé
Lors d'une entrée dans le champ Adresse, les champs Masque, Site et Subnet sont automatiquement remplis.

Si l'adresse IP est temporaire, cocher la case dans la colonne Temporaire.
Ceci fait apparaître une nouvelle colonne jusqu'à permettant de choisir une date de résiliation
Pour cela cliquer sur qui ouvre un calendrier où vous pouvez sélectionnez une date

Si l'adresse IP sera migré, cocher la case dans la colonne Migration
Deux colonnes aparaissent : le pour choisir une date de migration avec et Adresse IP future pour rentrez la future adresse IP
Comme pour le champ adresse IP, masque, site et subnet de migration seront automatiquement remplis.
Un icône apparaît vous permettant de visualiser ces informations ainsi que de savoir si l'adresse IP existe déjà ou non.
Une réservation de la future adresse portant le libellé 'Reservation de XXX.XXX.XXX.XXX' est également ajouté dans la table. Cet enregistrement est de type 'dynamic' (non modifiable).
Il est supprimé en même temps que la supression ou la migration de l'adresse IP.

La gestion des lignes permet:

  • l'ajout d'une ligne à la fin du tableau
  • la suppression d'une ligne
  • la suppression de toutes les lignes

Afin d'ajouter/modifier votre enregistrement, cliquez sur
Si certaines valeurs obligatoires manquent ou si certaines valeurs sont erronées, une fenêtre vous avertit et la requête n'est pas exécutée.
Vous pouvez alors effacer les lignes dont le résultat est correct et retravailler les lignes en erreurs.
Si aucune erreur n'est détectée, les listes ajout/modification et adresses IP se réinitialisent.
