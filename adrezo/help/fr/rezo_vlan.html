Gestion des Vlans

Avec les droits nécessaires, vous pouvez gérer les vlans.

Vous trouverez dans la page deux zones :
  • une zone d'ajout/modification
  • la liste des vlans existants avec leurs informations
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un vlan en cliquant sur
  • Modifier un vlan en cliquant sur
En cas d'ajout ou de modification, entrez ou modifiez les différentes informations puis cliquez sur pour valider.
Vous pouvez également cliquer sur pour annuler, tous les champs redeviennent vides.
Un vlan est défini par plusieurs champs :
  • le site de rattachement
  • l'id du vlan
  • son libellé
La création d'un site entraîne automatiquement la création du vlan 0 qui signifie 'Pas de Vlan'
