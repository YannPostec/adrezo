Préférences Utilisateur

Vous pouvez dans cette page modifier vos préférences.
Elles sont composées du :

  • choix de la langue de l'application
  • choix du contexte initial sur lequel vous arrivé en vous connectant
  • changement de votre mot de passe si votre compte est un compte local
  • Activer/Désactiver la fonction d'affichage d'urls quand vous cliquez sur une adresse IP
  • Activer/Désactiver la fonction d'affichage du vendor quand vous cliquez sur une adresse MAC
  • Modifier le timeout du slideshow
