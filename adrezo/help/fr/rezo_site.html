Gestion des Sites

Avec les droits nécessaires, vous pouvez gérer les sites.

Vous trouverez dans la page deux zones :
  • une zone d'ajout/modification
  • la liste des sites existants avec leurs informations
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un site en cliquant sur . Attention cela entraine la suppression de tous les objets associés
  • Modifier un site en cliquant sur
En cas d'ajout ou de modification, entrez ou modifiez les différentes informations puis cliquez sur pour valider.
Vous pouvez également cliquer sur pour annuler, tous les champs redeviennent vides.
Un site est défini par plusieurs champs :
  • le contexte de rattachement
  • le code site
  • son libellé
La création d'un site entraîne automatiquement la création du vlan 0 associé à ce site qui signifie 'Pas de Vlan'
