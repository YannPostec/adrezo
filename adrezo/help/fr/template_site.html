Gestion des Modèles de sites

Avec les droits nécessaires, vous pouvez gérer les modèles de sites.

On distingue deux types de modèle de site

  • Standard : Listes définies de vlans. Subnets englobés dans un surnet. Ce modèle a un masque supérieur à 1
  • Non-Standard : Liste définie de vlans. Subnets à trouver dans une liste de surnets. Pas de surnet global. Ce modèle a un masque égal à 0

Vous trouverez dans la page deux zones :
  • une zone d'ajout
  • la liste des modèles existants avec leurs informations
Pour chaque ligne de la liste, vous pouvez :
  • Effacer un modèle en cliquant sur . Attention cela entraine la suppression de tous les objets associés
  • Modifier un modèle en cliquant sur
En cas d'ajout, entrez les différentes informations puis cliquez sur pour valider.
En cas de modification, modifiez les différentes informations puis cliquez sur pour valider.
Dans les deux cas, vous pouvez cliquer sur pour annuler et revenir aux valeurs initiales.
Un modèle de site est défini par plusieurs champs :
  • son nom
  • son masque
  • ses vlans
  • ses subnets
La création d'un site entraîne automatiquement la création du vlan 0 associé à ce site qui signifie 'Pas de Vlan'

Il est possible de n'afficher dans le tableau que les lignes contenant la chaîne de caractères rentrée dans le champ Rechercher

Afin de configurer les vlans ou les subnets d'un modèle de site, il suffit de cliquer sur dans la colonne appropriée.
