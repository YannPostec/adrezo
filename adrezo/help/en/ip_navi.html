Zone : Search Criteria

This zone allow you to build an SQL search chain.
Once the chain is built, you apply using
The text area shows the search chain, it is not editable

Form : Creating search chain

5 differents types of elements could be added :
  • Open Parenthesis
  • Close Parenthesis
  • Logical element AND
  • Logical Element OR
  • Search criteria
A search criteria is an expression composed of three elements:
  • A search key to choose in list
  • An operator to choose in a key-specific list. Example for a numeric value
  • A search value to choose in a list or to type depending on the key

Adding a criteria bu clicking on
The four other elements help to build a logical expression and you add them by clicking on their respective buttons.

Adding those elements have constraints

  • A ( can be placed only after AND, OR or at the start of the chain
  • A ) can be placed only after a search criteria
  • AND, OR can be place only after a ) or a search criteria
  • A search criteria can be placed only after a ) or another search criteria
If thos rules are not followed, the elemet is not added

Delete Buttons and delete respectively the full chain or only the last added element

Zone : Sort Criteria

This zone allow you to apply a sort key on some columns of IP address list.

Choice is made with a list
By selecting the proper radio button, the sort order could be Crescent (Default) or Decrescent.

Every change on a criteria is immediately take into account

Zone : Excel Export

You could have a database export under excel format by clicking on with 3 variants :

  • a complete export of IP addresses regardless of selected context
  • a export of only IP addresses for the selected context
  • a export of only IP addresses restricted by the current search chain
