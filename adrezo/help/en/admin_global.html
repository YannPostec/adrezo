Application Management

With proper permissions, you could access application settings.
Settings are :

  • Application default language
  • Application default context
  • Admin account management
  • IP Addresses URL management
  • Maintenance tools

Default choices

Select language or context by clicking on it.
Current language or context appears in green.

Admin account

Here you could change password and/or mail of the admin account.
Password is never displayed.

URLs Management

Each url is built with <protocol>://<IP>:<port>/<URI>
Port and URI are not mandatory, if you enter port 80 for protocol http or 443 for https, port is not taken into account.

Link between exploitation and normal accounts

With external authentication, if accounts are exploitation type without mail address,
And if their logins are derived from the normal account login, which contains a mail address,
You can link this mail information by typing prefix and/or suffix used to retrieve normal account login.

Maintenance Tools

Display application logs :

  • photo : errors from photos processing
  • database : errors from database
  • adrezo-default : application error log
  • adrezo-api : api log
  • adrezo-beans : beans log
  • adrezo-jsp : jsp taglib log
  • adrezo-servlets : servlets log
  • adrezo-listener : listeners log
  • quartz : quartz module log in charge of scheduled tasks
  • quartz-jobs : scheduled tasks log

If configured, test connection to Cacti MySQL database

Logo Management

You could change the login page logo by uploading an image in PNG format.
You could also switch back to the default logo by clicking on the right button.
