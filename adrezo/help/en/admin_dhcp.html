DHCP Servers Management

With proper permissions, you can manage DHCP servers used by the scheduled task gathering the DHCP informations.

On this page, you will find two zones :
  • An Add/Modify zone
  • the list of existing servers with their informations
Within each line, you could :
  • Delete a server when you click on
  • Modify a server when you click on
When adding or modifying, enter or modify informations then click on to validate.
You could also click on to cancel, going back to empty values.
You could also manage, server by server, scopes which will not be taken into account during the gathering of informations :
  • Add an exclusion when you click on
  • List and delete exclusions when you click on
Deletion is processed by clicking on
Within the add box, a server scope list is provided (without the already excluded), you could add an exclusion by clicking on next to the selected scope.
You could test the connection to a DHCP server by clicking on
A server is defined by many fields :
  • DHCP server type
  • server hostname or ip
  • port used to access the API on the server
  • the access method: SSL (HTTPS) or not (HTTP)
  • if you use Basic Web Authentication to connect to the server
  • if authentication, the login used
  • if authentication, the password used (this password is never displayed)
  • list of its excluded scopes, if any
