Add IP address

Mandatory Fields are Name and IP Address, it's better to fill the description
When you enter a IP Address, fields Mask, Site and Subnet are automatically filled.

If this IP address is temporary, check the box in the Temporary column.
A new colum appears until where you can choose an end date
Click on to select a date

If this IP address will be migrated, check the box in the Migration column.
Two columns appears : the to choose a migration date with and Upcoming Address to enter the future IP address
As for the IP address field, the migrating mask, site and subnet will be automatically filled.
An icon appears on this you can view those informations and if this IP address is already existing or not.
A reservation of future IP address with description 'Migration de XXX.XXX.XXX.XXX' is also added to the database. This record is of type 'dynamic' (non alterable).
It is deleted when you delete or migrate this IP address.
