IP addresses migration

You'll find the Column "Migrate" in the IP addresses list.
In this colum, on each line where a migrating IP address is present, there is a checkbox
Select records to migrate by selecting thoses checkboxes.
On the column header there is a All link which invert the selection you made.

Click afterwards on which display a confirmation window.
After confirmation, selected records will be migrated and the list will be refreshed.
