Search Available IP addresses

This section search in a subnet available IP addresses (in database, not in real network life), you could start the search at the beginning or at a defined address.

Mandatory fields are number of addresses (default 1) and subnet.
The starting IP is optional. By default, search start at the beginning of the specified subnet.
If entered, starting IP must be in the subnet specified.

Start the search by clicking on

If IP are available, result takes form in a table with :

  • IP address
  • possiblity to send the address to a line in the Add/Modify section
  • possiblity to send the address in migrating field of a line in the Add/Modify section if migration is checked

All lines of the result set could :

  • be added at the end of the Add/Modify section
  • replace totally the Add/Modify section array
Those actions clear the search once finished.

You could manually clear the search by clicking on or search again
