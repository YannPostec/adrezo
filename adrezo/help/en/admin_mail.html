Mail Management

With proper permissions, you can modify the content of mails sent by the application.

On this page, you will find a list of mails with :
  • Number of this template
  • Language of this template
  • Database table used to build the variables
  • Mail recipient
  • Mail subject
  • Mail message
In subject or message, variables could be placed. They will be replace with values from the pointed out table.
Variables names are this table columns names.
Thos variables must be placed between #. Example : #IDX# display value from IDX column.

For some entries, recipient, subject or message could be replaced with USERDEF, they will be managed by the application
For some templates as the one sending mail to user, the database table will not be used

After modification, click on to validate.
