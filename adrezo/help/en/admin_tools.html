Administration Tools

With proper permissions, you have access to some administration tools.
They are :

  • Temporary IP Addresses
  • Migrating IP Addresses

Temporay IP Addresses

Display every temporary IP addresses.
In addition to informations on this IP address, you have the user who set it and the end of validity date.
The date is in red if in past.

Migrating IP Addresses

Display every migrating IP addresses.
In addition to informations on this IP address, you have the user who set it and the date planned for migration.
The date is in red if in past.
