Furnitures Type Management

This page display by default the context main stock.
You can select a site with the scroll list or return to the main stock with
This page display something only if the main stock is defined for the current context.

With proper permissions, you can manage furnitures types.

On this page, you will find a table with :
  • An Add zone on the first line
  • the list of existing sites with their informations on the next lines
Within each line, you could :
  • Delete a type when you click on . Be careful, it will delete the movement history.
  • Modify a type when you click on
When adding or modifying, icons appears.
Enter or modify informations then click on to validate.
You could also click on to cancel, going back to initial values.
A type is defined by many fields :
  • its category
  • the index of case or shelf in the stock
  • its description
  • the threshold which send a mail to the stock administrator when triggered
  • the number of elements in ongoing order
Threshold is triggered when stock plus ongoing order are less then this threshold.
