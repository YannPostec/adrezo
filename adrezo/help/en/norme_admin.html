Network norm Management

With proper permissions, you can manage surnets defining the norm.
Use to enter or leave admin mode.

On each surnet, you find four icons :

  • Delete the surnet
  • Add a child surnet to this surnet
  • Modify surnet informations
  • Manage the subnet list of this surnet
  • Compute available subnets in this surnet

Delete

Clicking on it display a confirmation window
Be careful, it's a recursive delete within all subnets/surnets attached. However, there is no modifications to subnets

Add

Clicking on it display a window where you enter informations on the surnet child :
  • its IP
  • its mask
  • its description
  • its informations in a text editor
  • if surnet permit to launch the subnet calculator
Validate by clicking on
If IP/mask you entered don't make a valid subnet address, a warn message appears.
However, it is still possible to add a surnet with invalid subnet address.

Modify

In the same way that adding a surnet, a window is dispayed with the same informations

Manage Subnets List

This list represent the subnets attached to this surnet.
In the right area are the attached subnets.
In the left area are the subnets you could attached.
The left area is filled each time you choose a context, you could reduce the choice by selecting a site.
You move the subnets between the areas using the icons :
  • From left to right
  • From right to left
To attach the subnets defined to the surnet, click on

Available Subnets Calculator

In the opening box, you just have to enter the desired mask of subnets you search.
After validation, a list of available subnets will be displayed.
