This page display something only if the main stock is defined for the current context.

This page display the stock sorted by type.
Each type has many informations :

  • its category
  • the index of case or shelf in the stock
  • its description
  • current stock (in red when less than the threshold defined if any)
  • the threshold which send a mail to the stock administrator when triggered
  • the number of elements in ongoing order

You can consult movements history for a type by clicking on
This will display a window with the movements list sorted by descrescent date.
By default, the last 20 movements are displayed, you can change this value at the top of the page.
